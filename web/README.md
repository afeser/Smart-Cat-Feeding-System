## Steps to deploy locally:

* `git clone --single-branch --branch web https://github.com/afeser/finalproject/git`
- `cd finalproject/web`
+ `python3 -m venv venv`
- `source venv/bin/activate` or on Windows: `venv\Scripts\activate`
+ `pip install -r requirements.txt`
* `flask run`
